# M4XDEV\.com

Personal website with the goal of being a joy to work on with minimal maintenance.

## Project setup
1. Install dependencies
```bash
$ pnpm install
```

2. Copy example env file
```bash
$ cp .env.example .env
```

## Running the project
```bash
# Development
$ pnpm dev

# Production
$ pnpm start
```

## Development notes
1. Verifying types
```bash
$ pnpm types:check
```
